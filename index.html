Otázky a odpovede

Čo je QA Academy?

QA Academy je moderný kurz zameraný na testovanie softvéru.

Cieľom kurzu je pripraviť účastníkov na softvérové testovanie aplikácii a uľahčiť im vstup do sveta IT.

Čo musíš vedieť?

Uchádzač o kurz by mal ovládať základy práce s počítačom, anglický jazyk

Kedy bude kurz prebiehať?

Od 2.4.2019 do 30.5.2019

Kde bude kurz prebiehat?

Bratislava

Čo sa na kurze naučíš?
  • Základy testovania
  • Techniky tvorby testov
  • Session based testing
  • Základné architektúry systémov
  • HTML, CSS, Javascript
  • JIRA
  • Postman
  • Testovanie API
  • Základy automatizovaného testovania
  • SQL
Čo je v cene kurzu?
  • 16 lekcií x 3 hodiny 2x týždenne
  • certifkácia
  • HR workshop a poradenstvo
Cena kurzu

599,- s DPH
